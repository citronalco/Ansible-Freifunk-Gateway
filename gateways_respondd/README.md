Diese Rolle installiert mesh-announce von https://github.com/ffnord/mesh-announce.git

mesh-announce stellt Informationen über das Gateway auf Port 1001 per Broadcast zur Verfügung, so wie es wie es respondd auf den Access Points macht.
Diese Informationen können dann von Hopglass, Yanic oder ähnlichen Programme ausgewertet werden, um z.B. die Gateways auf einer Knotenkarte anzuzeigen.

Es ist keine Konfiguration nötig.
